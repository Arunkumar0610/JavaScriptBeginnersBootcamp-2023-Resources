document.querySelector();-the document method querySelector() returns the first element with in the document that matchs the specified selector or group of selectors.
- if you are targeting the class it will be the first class in the DOM that matches the value that you are looking for.
- if you are targeting the id then your first possible match will be the only match because there should be an ID of only one value per page.

-lets assign the id title to var title1
=>var title1= document.querySelector('#title'); so the element with the id title is the only match that gets assigned to the title 1 variable
- notice that we had to add the hash symbol(#) as we would if we were targeting something with CSS. This is because the querySelector() DOM method is more generic than the getElementById() that is only looking for IDs.
- querySelector can target any element on our page. so, we have to specify what kind of element we are looking for.

=> if we have both class and id with name title than the querySelector() method it will get title tag element. if no title tag is there it will give nulldocument.querySelector('title');
- document.querySelector('.title'); - will get the first element with class title.
- document.querySelector('#title');- will get the id with that element

More DOM methods and properties- appendChild(),removeChild(),createElement() and firstChild

appendChild()-> adds a node to the end of the list of children of specified parent node.

removeChild()-> removes a child node from the DOM

createElement()-> creates the HTML element specified by tagName

property - firstChild-> read-only property returns the node's first child in the tree

localStorage-> web storage that stores data with no expiration date

run localStorage; in console will get o/p of storage object ->Storage{length:0}

Four Methods that used with localStorage are setItem(),getItem(),removeItem(),clear()

setItem()-> add a key and value to localStorage.
localStorage.setItem('todos','Make Breakfast');
localStorage.setItem('doNotDo','Sleep in class');
localStorage;
o/p->Storage{doNotDo:"Sleep in class",todos:"Make Breakfast",length:2}

getItem()-> retrieve a value by the key from localStorage.
localStorage.getItem('todos');
o/p-> "Make Breakfast"

removeItem()-> removes an item from localStorage by using key.
localStorage.removeItem('doNotDo');
localStorage;
o/p->Storage{todos:"Make Breakfast",length:1}

clear()-> clears all storage of localStorage.
localStorage.clear();
localStorage;
o/p->Storage{length:0}
