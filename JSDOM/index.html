DOM & Nodes

There are so many nodes but we mainly use three nodes they are:

  • Element Nodes - When an HTML element is an item in the DOM. It is referred to as an Element Node
  • Text Nodes - Any alone text outside of an element is known as text node
  • Comment Nodes- an HTML comment is an comment node
In addition to these nodes the document itself is an document node. Which is the root of all other nodes

DOM Tree: The document consists of a tree structure of nested nodes, which is often referred to as the DOM tree.

Like in the Family tree the nodes in the dom are also referred to as parents, children And siblings, depending on their relation to other nodes.
